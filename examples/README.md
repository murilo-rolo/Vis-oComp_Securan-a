# Examples

Esta pasta contém scripts de exemplo demonstrando como usar os diferentes módulos do projeto.

## Scripts Disponíveis

- **example_usage.py** - Exemplo básico de uso do modelo ResNet-LSTM
- **example_cnn3d_usage.py** - Exemplos de uso do módulo CNN 3D
- **example_emotion_usage.py** - Exemplos de uso do módulo EmotionNet
- **example_evaluation_usage.py** - Exemplos de avaliação de modelos
- **example_multimodal_usage.py** - Exemplos de uso do modelo multimodal
- **example_pose_usage.py** - Exemplos de extração e uso de pose
- **example_realtime_usage.py** - Exemplos de inferência em tempo real

## Como Usar

Execute qualquer script de exemplo diretamente:

```bash
python examples/example_usage.py
```

Ou execute a partir da raiz do projeto:

```bash
cd examples
python example_usage.py
```

## Nota

Estes são scripts de exemplo educacionais. Para uso em produção, use os scripts principais na raiz do projeto (`train_pipeline.py`, `run_*.py`, etc.).
