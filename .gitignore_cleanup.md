# Limpeza do Git - Remover Imagens e Vídeos do Índice

## ⚠️ IMPORTANTE

Este arquivo contém instruções para remover imagens e vídeos do índice do Git sem deletá-los do disco.

## 🔧 Comandos para Executar

Se você já adicionou imagens/vídeos ao Git antes de atualizar o .gitignore, execute:

```bash
# Remover imagens do índice do Git (mantém no disco)
git rm --cached dataset/**/*.jpg
git rm --cached dataset/**/*.jpeg
git rm --cached dataset/**/*.png
git rm --cached dataset/**/*.gif
git rm --cached dataset/**/*.bmp
git rm --cached dataset/**/*.tiff
git rm --cached dataset/**/*.webp

# Remover vídeos do índice do Git (mantém no disco)
git rm --cached dataset/**/*.avi
git rm --cached dataset/**/*.mp4
git rm --cached dataset/**/*.mov
git rm --cached dataset/**/*.mkv
git rm --cached dataset/**/*.flv

# Ou remover tudo de uma vez (mais eficiente)
git rm -r --cached dataset/

# Depois adicionar apenas os arquivos permitidos (CSV, TXT, etc.)
git add dataset/**/*.csv
git add dataset/**/*.txt
git add dataset/**/*.json
git add dataset/**/*.md
```

## 📝 Nota

- `--cached` remove do índice do Git mas mantém os arquivos no disco
- Os arquivos continuarão existindo localmente
- Eles não serão mais rastreados pelo Git
- O .gitignore agora previne que sejam adicionados novamente
